### 简介

打车散点图层是通过 L7 的 pointLayer 图层去渲染完成的。

用户可以通过配置 `shape` 参数来选择点的类型，也可通过其他 API 去配置打车散点样式等。

详情可见[L7 PointLayer](https://l7.antv.antgroup.com/api/point_layer/pointlayer)
