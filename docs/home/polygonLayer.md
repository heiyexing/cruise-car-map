### 简介

热点区域图层是通过 L7 的 PolygonLayer 图层去渲染完成的，通过传入字段来显示当前区域乘座热度

用户可以通过配置 `shape` 参数来选择多边形的数据的类型，也可通过其他 API 去配置热点区域图层样式等。

详情可见[L7 PolygonLayer](https://l7.antv.antgroup.com/api/point_layer/polygonlayer)
